=====================Programming=====================
To run this program, import the SCserver and SCclient folders to Eclipse.
Run the SCclient first, the console will show 'waiting for connection...'.
Then run the SCclient, there will be a calculator GUI appear.
All designs are following the rubrics and requirements.
After each successful calculation, the server will output the previous successful calculation and the total number so far.
